# Security policy

## Supported version

Security fixes are applied to the default branch of this ROS 2 Humble port.

## Reporting a vulnerability

Please do not disclose a suspected vulnerability in a public issue before it is assessed. Use GitHub's private security advisory feature for this repository when available, or contact the repository maintainer privately through GitHub.

Include a clear description, affected files or packages, reproduction steps, impact and any suggested mitigation. You should receive an acknowledgement within seven days.
